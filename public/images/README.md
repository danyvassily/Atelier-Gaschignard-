# Structure des images dans public/

Toutes les images sont maintenant organisées dans le dossier `public/images/` :

## Images principales
- `logo.png` - Logo de l'Atelier Gaschignard
- `hero.jpg` - Image hero de la page d'accueil
- `page-accueil.jpg` - Image alternative page d'accueil
- `contact-hero.jpg` - Image hero de la page contact
- `contact.jpg` - Image alternative contact
- `services-hero.jpg` - Image hero de la page services
- `services.jpeg` - Image alternative services

## Photos food
Toutes les photos de créations culinaires sont dans `public/images/food/` :
- 22 images au total
- Format organisé pour la galerie

## Utilisation dans le code
Les images sont référencées avec le chemin `/images/...` dans les composants Next.js.

## Note
Toutes les images proviennent du dossier source "Photos d_illustration" et du logo fourni.
Aucune image générée par IA n'a été utilisée.
